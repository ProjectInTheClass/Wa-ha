## ADS

- 누구를 위한 : 로토스코핑을 하려고 하는 모션그래픽 디자이너
- 어떤 기능의 : 촬영하거나 불러온 영상을 12,15,30프레임 이미지 시퀀스로 불러온 뒤, 그 위에 그림을 그릴 수 있게 만든다.
- 어떤 차별점의 : 영상을 따로 이미지 시퀀스로 렌더한 다음 불러오는게 아니라, 앱 자체에서 바로 이미지로 만들어준다. 따라서 로토스코핑 작업에 시간을 절약할 수 있다.
